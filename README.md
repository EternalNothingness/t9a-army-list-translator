# T9A Army List Translator

A simple tool to translate army lists for the tabletop game T9A.

## Dependencies

flex, make

## Building Instructions

Run `make` in the src directory after installing all necessary dependencies.

## Usage

Let armylist.txt contain the armylist to translate. Run `cat path/to/armylist.txt | path/to/scanner` to translate it.

## Extensions

If your army has not been translated yet, you can add the necessary rules in the rules directory.
